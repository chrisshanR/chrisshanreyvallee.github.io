Spot The Fire

A mobile application which is created by me and my team during our participation in NASA Hackathon on 2019. The mobile application was created by utilizing android studio, flutter and firebase cloud database. The application will enable user to report fire incident to the fire departments and the application would also be able to deliver notification to all registered user mobile phone if there is fire incident nearby. Furthermore, the mobile application have several other functionalities such as fire incidents database, self support guide during fire, emergency call connected to default phone call application and a fire map. A fire map is a map that shows the likelihood of fire incident happening around the world in real time which is the fire map provided directly by NASA.
